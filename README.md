# Echo App - POC

## Description
Création d'une application web & mobile, permettant à un utilisateur de renseigner quotidiennement à l'écrit ou à l'oral son ressenti journalier. Ce ressenti est analysé par l'IA pour comprendre au mieux son état émotionnel et créer une visualisation correspondante.  
L'application prend en compte les analyses et représentations passées pour faire une rétrospective mensuelle et annuelle, avec une analyse textuelle de l'évolution au fil du temps.

## Fonctionnalités à développer
1. Capture d'émotions  
2. Analyse IA émotionnelle  
3. Génération visuelle algorithmique  
4. Mosaïque interactive  
5. Statistiques & insights  
6. UX/UI premium
